## Coding Schema Summary

The coding schema captures 43 variables per holding:

- **Q1-Q4**: Case metadata (ID, date, chamber, holding count)
- **Q5-Q9**: Holding identification and GDPR concepts
- **Q10-Q20**: Interpretive methods and sources
- **Q21-Q32**: Balancing and necessity analysis
- **Q33-Q43**: Outcome and ruling direction

See `coding-agent-instructions.md` for complete definitions and coding rules.
